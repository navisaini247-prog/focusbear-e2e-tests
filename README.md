# Focus Bear E2E Tests

Automated end-to-end testing pipeline using GitHub Actions.

## CI Status
![CI](https://github.com/navisaini247-prog/focusbear-e2e-tests/actions/workflows/e2e-tests.yml/badge.svg)

## Features
- Automated Mac testing
- Python test execution
- HTML test report generation
- CI/CD pipeline with GitHub Actions

## Workflow
Tests automatically run when:
- Code is pushed to `main`
- A Pull Request is created

## Report
The workflow generates an HTML report that can be downloaded from the Actions artifacts.
